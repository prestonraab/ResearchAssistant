## Papers to Add Manually

The following papers need to be added to Zotero. Use one of these methods:

### Method 1: Zotero Browser Connector (Recommended)
1. Open the paper URL in your browser
2. Click the Zotero browser connector icon
3. Verify the metadata is correct
4. Add to your "BookChapter" collection

### Method 2: Manual Entry
1. Open Zotero
2. Create new item → Journal Article
3. Fill in the metadata from the details below
4. Add to "BookChapter" collection

### Method 3: DOI Import
1. In Zotero, click the green plus icon → "New Item by Identifier"
2. Enter the DOI
3. Verify and save
4. Add to "BookChapter" collection
